# Arquitectura de Software – API Documentation

## Nombre
Paola Valeria Burgos Pomacusi

## Descripción
Este repositorio contiene la documentación técnica y el diseño arquitectónico
de un sistema API, gestionado mediante Git y GitHub como parte del módulo Arquitectura de Software y Gestion de Repositorios.

## Contenido
- Definición de endpoints REST
- Esquema GraphQL
- Diagrama de arquitectura del sistema

## Metodología
El trabajo se organiza mediante Issues, ramas independientes y Pull Requests,
siguiendo buenas prácticas de control de versiones.
